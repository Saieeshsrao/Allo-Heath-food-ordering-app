
## Run Locally

Clone the project

```bash
  git clone https://github.com/Saieeshsrao/Allo-Heath-food-ordering-app.git
```

start your MongoDB compass
edit the .env file according to the Db url

Go to the backend
```bash 
cd backend
```

```bash
  npm i
  npm run dev
```

Go to the frontend

```bash
  cd frontend
```
```bash
npm i
```
```bash
npm start 
```
